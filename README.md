# STR GIMS Enterprise

## Spatial Tiger Reserve - Geographic Information Management System

STR GIMS Enterprise is a web-based GIS application for managing wildlife, plant inventories, forest ranges, and spatial data.

## Technology Stack

- PostgreSQL 18.4
- PostGIS
- PostgREST
- GeoServer 2.28.4
- HTML5
- CSS3
- JavaScript
- Leaflet

## Features

- Secure Login
- Plant Inventory
- Animal Inventory
- Spatial Search
- GeoServer Integration
- Dashboard
- Reports
- User Management
- Audit Logs
